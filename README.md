# 💬 Real-Time Chat Application

## 🚀 Overview

This is a full-stack real-time chat application built using MERN stack. It allows users to communicate instantly with features like authentication, live messaging, and responsive UI.

---

## 🛠️ Tech Stack

* Frontend: React.js, Tailwind CSS, Zustand
* Backend: Node.js, Express.js
* Database: MongoDB
* Real-time: Socket.IO
* Cloud: Cloudinary (for image upload)

---

## ✨ Features

* 🔐 User Authentication (Login / Signup)
* 💬 Real-time messaging using Socket.IO
* 🟢 Online/Offline user status
* 📸 Image sharing support
* 🎨 Modern responsive UI

---

## 📁 Project Structure

```
backend/   → API & server logic  
frontend/  → UI & client side  
```

---

## ⚙️ Setup Instructions

### 1. Clone repo

```
git clone https://github.com/your-username/real-time-chat-app.git
```

### 2. Install dependencies

```
cd backend && npm install
cd ../frontend && npm install
```

### 3. Add environment variables

Create `.env` in backend:

```
PORT=5000
MONGO_URI=your_db_url
JWT_SECRET=your_secret
```

### 4. Run project

```
# backend
npm run dev

# frontend
npm run dev
```
